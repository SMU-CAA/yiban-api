## yiban EGPA tool
[![Releases](https://img.shields.io/github/downloads/simonsmh/yiban-api/total.svg)](https://github.com/simonsmh/yiban-api/releases)

易班EGPA工具是使用Python语言构建的开源的工具，使用yiban-api调用网页接口，使用PyQt5作为界面设计，实现方便的自动化易班使用功能。
具体功能在软件中有介绍。
